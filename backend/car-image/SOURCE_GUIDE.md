## ─────────────────────────────────────────────
## ⑥ STEP-BY-STEP WORKFLOW TO FILL ALL FOLDERS
## ─────────────────────────────────────────────

### Step 1 — Priority order (easiest first)
Start with brands that have great press portals:
1. Hyundai, Kia, Toyota, BMW, Audi, VW, Ford (best press portals)
2. Mahindra, Maruti Suzuki, Renault, Nissan (good Indian market sources)
3. BYD, Chery, Haval, Geely, Maxus (Chinese brands — use global sites)
4. Deepal, Riddara, Proton (newer/smaller — use official websites directly)

### Step 2 — Download images
- Download from press portals in bulk (ZIP files)
- Aim for: white background, 1200×800px minimum
- Save originals first, rename after

### Step 3 — Rename using the manifest
Open `IMAGE_MANIFEST.txt` — it has the exact filename for each slot.
Example: `audi-q3-exterior-front-white-bg.jpg`

### Step 4 — Compress before uploading
Use TinyPNG (tinypng.com) — drag and drop up to 20 images free.
Target: under 200KB per image for web use.

### Step 5 — Drop into the correct folder
Each folder already exists and is waiting for the real image.
Just replace the empty placeholder `.jpg` files with real images.

One-time setup (5 min):
1. Open any Google Sheet (create one named CarDatabase)
2. Go to Extensions → Apps Script
3. Delete all code and paste the contents of apps-script.gs
4. Click Deploy → New deployment → Web app
5. Set "Who has access" to Anyone → Deploy
6. Copy the Web app URL and paste below ↓
